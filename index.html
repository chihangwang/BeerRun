ABOUT BEER RUN

10/19 馬場町河濱公園

想要在揮汗路跑之後，痛快暢飲冰涼的青島啤酒嗎？快加入BEER RUN – 全世界醉HIGH的路跑，和比爾(BEER)先生及青島啤酒一同狂奔、一同狂飲、一同狂歡！！

BEER RUN 全長3.2公里，途中青島啤酒突襲軍為你噴灑冰涼啤酒，讓你補給再出發，還可以划酒拳、推酒杯闖關贏啤酒，賽後派對瘋狂啤酒大賽、Live Band、舞蹈表演等著你！快來一起開瓶，享受全世界醉HIGH的BEER RUN吧！

前500名比爾，贈安全第一喝啤酒帽一頂(價值420元)！快點一起來報名，衝刺吧比爾！！！
